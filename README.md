## 📌 Overview

This is my **personal portfolio website** built to highlight my projects, technical skills, and contact details. It is designed with a clean UI, responsive layout, and smooth animations — making it suitable for desktop and mobile screens.

Whether you're a recruiter, collaborator, or a fellow developer, this site gives a quick insight into who I am, what I build, and how to reach me.
